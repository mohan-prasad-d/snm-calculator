Understanding the Method
Divided Difference Formula Core
f[xi,xj] = (f[xj] − f[xi]) / (xj − xi)
Higher orders: f[xi,...,xi+k] = (f[xi+1,...,xi+k] − f[xi,...,xi+k−1]) / (xi+k − xi)
Newton's Polynomial Interpolation
P(x) = c0 + c1(x−x0) + c2(x−x0)(x−x1) + ···
Coefficients: ck = f[x0, x1, ..., xk] — the diagonal of the DD table.
The polynomial passes exactly through all given data points.
Key Properties Reference
Degree = n−1  (for n data points)
Divided differences are symmetric in their arguments.
If f(x) is a polynomial of degree k, then (k+1)th DD = 0.
Horner's method gives efficient evaluation.
Newton's Divided Difference Formula
Goal: Estimate f(x) from known data pairs (xi, yi)
Algorithm:
  1. Build divided difference table[i][j]
  2. Extract coefficients: ck = table[0][k]
  3. Evaluate: P(x) = c0 + c1(x−x0) + ...
  4. Result: f(x) ≈ P(x)
