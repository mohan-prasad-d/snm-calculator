Method Overview
MILNE PREDICTOR FORMULA
yn+1(p) = yn−3 + (4h/3)[2fn−2 − fn−1 + 2fn]
where fn = f(xn, yn)
MILNE CORRECTOR FORMULA
yn+1(c) = yn−1 + (h/3)[fn−1 + 4fn + f(xn+1, yn+1(p))]
where yn+1(p) is the predicted value from above
ITERATION PROCEDURE
①  Compute f₀,f₁,f₂,f₃ from given starting values
②  Apply Predictor: compute yn+1(p)
③  Evaluate f(xn+1, yn+1(p))
④  Apply Corrector: compute yn+1(c)
⑤  Use corrected value as yn+1 and repeat
Properties of Milne's Method:
▶ Fourth-order accurate: error ∝ h⁴
▶ Multi-step: uses the last 4 points
▶ Requires a separate starter (e.g. RK4)
▶ Two evaluations per step (predict + correct)
KEY INSIGHT
The predictor gives an estimate; the corrector refines it using the predicted slope at the new point, achieving higher accuracy than single-step methods.
