Method Overview
PHASE 1 — RK4 STARTING VALUES
k₁ = h · f(xₙ, yₙ)
k₂ = h · f(xₙ+h/2, yₙ+k₁/2)
k₃ = h · f(xₙ+h/2, yₙ+k₂/2)
k₄ = h · f(xₙ+h, yₙ+k₃)

yₙ₊₁ = yₙ + (1/6)(k₁ + 2k₂ + 2k₃ + k₄)
PHASE 2 — ADAMS–BASHFORTH FORMULAS
2-step:  yₙ₊₁ = yₙ + h/2 [3fₙ − fₙ₋₁]
3-step:  yₙ₊₁ = yₙ + h/12 [23fₙ − 16fₙ₋₁ + 5fₙ₋₂]
4-step:  yₙ₊₁ = yₙ + h/24 [55fₙ − 59fₙ₋₁ + 37fₙ₋₂ − 9fₙ₋₃]
How the calculation works:
① Use RK4 to compute the required starting values
② Evaluate f(xₙ, yₙ) at each known point
③ Apply weighted combination of previous slopes
④ Predict yₙ₊₁ using the explicit AB formula
⑤ Repeat using the newly computed point
KEY INSIGHT
Adams–Bashforth uses previous slopes to predict the next value
Higher-order methods use more past data for greater accuracy
