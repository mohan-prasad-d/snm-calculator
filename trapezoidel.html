How It Works
TRAPEZOIDAL RULE FORMULA
h = (b − a) / n     (step size = width of each trapezoid)
xi = a + i × h    (for i = 0, 1, 2, …, n)

∫ab f(x) dx ≈ h/2 × [f(x0) + 2f(x1) + 2f(x2) + … + 2f(xn-1) + f(xn)]

Coefficients: 1 for endpoints (x0, xn),   2 for interior points
How the calculation works:
① Divide [a, b] into n equal parts of width h = (b-a)/n
② Compute f(x) at each dividing point x0, x1, …, xn
③ Multiply: endpoints × 1, interior points × 2
④ Add all up, then multiply by h/2
⑤ Result = approximate area under the curve
ERROR ESTIMATE
|ET| ≤ (b−a)3 / (12 × n2) × max|f′′(x)|
Doubling n reduces the error by factor of 4 (quadratic convergence)
