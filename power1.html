Power Method Algorithm
Given A (n×n) and X₀:
for k = 1, 2, … n iterations:
  Yk = A · Xk-1
  λk = norm(Yk)
  Xk = Yk / λk
λ → dominant eigenvalue
Xn → dominant eigenvector
