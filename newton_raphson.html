Step Size Check ✓ Recommended
|xn+1 − xn| < ε
Stop when the gap between two consecutive answers is smaller than ε.
Example: if x₄ = 1.521379 and x₅ = 1.521380, the step = 0.000001 which is very tiny → stop.
Relative Change Check For large values
|xn+1 − xn| / |xn| < ε
Stop when the step is tiny relative to the answer's size.
Better when your root is a very large number (e.g. x ≈ 10000). Avoids false stops near zero.
Function Value Check Strictest test
|f(xn)| < ε
Stop when f(x) is practically zero itself.
Directly tests if the answer actually satisfies the equation. Most strict — may need more iterations.
Newton-Raphson Formula
Goal: Find x such that f(x) = 0
Repeat until converged:
  x0 ← your initial guess
  for n = 0, 1, 2, …:
    Compute f(xn) and f′(xn)
    xn+1 = xn − f(xn) / f′(xn)
    Check stopping rule above
    if condition met → stop & report root
