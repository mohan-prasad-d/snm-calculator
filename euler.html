Method Overview
EULER'S METHOD — CORE FORMULA
yₙ₊₁ = yₙ + h · f(xₙ, yₙ)
xₙ₊₁ = xₙ + h

where   h = step size  ·  f(x,y) = dy/dx  ·  n = step index
STEP-BY-STEP ITERATION
①  Given (xₙ, yₙ), evaluate f(xₙ, yₙ)
②  Multiply: slope = h × f(xₙ, yₙ)
③  Update: yₙ₊₁ = yₙ + slope
④  Advance: xₙ₊₁ = xₙ + h
⑤  Repeat from ① using the new point
Properties of Euler's Method:
▸ First-order accurate: error ∝ h per step
▸ Global error is O(h) — halving h halves error
▸ Explicit method — no system to solve
▸ Uses only one function evaluation per step
▸ May be unstable for stiff equations
KEY INSIGHT
Euler's method approximates the solution by following the tangent line at each point
Smaller h gives a more accurate approximation of the true curve
